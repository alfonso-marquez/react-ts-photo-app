## Tech Stack

**Frontend:** React+Typescript, Tailwind

## Installation

1. Clone this repo:

```
git clone https://github.com/alfonso-marquez/react-ts-photo-app.git
cd react-ts-photo-app
```

2. Install dependencies

```
npm install
```

3. Start Development Server

```
npm run dev
```
